# AI Coding Agent Rule


AI Agent는 코드 생성 전에 반드시:

1. Project Structure 확인

2. Existing Code 분석

3. Development Standard 확인


해야 한다.



## Code Generation Rule


새로운 Controller 생성 시:


확인:

- Service 존재 여부

- DTO 필요 여부

- Exception 처리 필요 여부



## Test Rule


기능 구현 후 반드시:


1. Unit Test 작성

2. Build 실행

3. Test 실행


결과 확인 후 완료한다.



## 금지


하지 말 것:


- Entity 직접 Response

- Controller Business Logic 작성

- Test 없는 기능 생성

- 기존 Architecture 무시
