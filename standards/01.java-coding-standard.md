# Java Coding Standard


## Naming Convention


### Class

PascalCase 사용

Example:

UserService

PaymentController



### Method

camelCase 사용

Example:

findUserById()



### Variable

camelCase 사용

Example:

userName



### Constant

UPPER_SNAKE_CASE 사용

Example:

MAX_RETRY_COUNT



## Code Style


- 의미 있는 변수명 사용
- 하나의 Method는 하나의 책임
- 불필요한 주석 작성 금지
- 중복 코드 최소화
