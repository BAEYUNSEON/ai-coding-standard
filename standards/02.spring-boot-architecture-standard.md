# Spring Boot Architecture Standard


## Package Structure


com.company.project


controller

service

repository

entity

dto

exception

config



## Controller Rule


Controller 역할:

- HTTP Request 처리
- Parameter Validation
- Service 호출


금지:

- Business Logic 작성
- Repository 직접 호출



Example:


@RestController
public class UserController {


    private final UserService userService;


}



## Service Rule


Service 역할:

- Business Logic 처리
- Transaction 관리


Example:


@Service
@Transactional
public class UserService {


}



## Repository Rule


Repository 역할:

- Database Access 담당
- Query 처리


금지:

- Business Logic 작성
